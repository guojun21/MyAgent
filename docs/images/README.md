## 📝 如何添加图片

### 方法1：手动复制
```bash
# 将架构流程图复制到这个目录
cp /path/to/your/image.png docs/images/architecture-flow.png
```

### 方法2：PowerShell复制
```powershell
Copy-Item "源图片路径" -Destination "docs/images/architecture-flow.png"
```
