## 📁 文件操作类（6个）

### 1. 基础文件列出
```

```
**预期**：调用 `list_files`，显示所有.py文件

```

```
**预期**：调用 `write_file`，创建文件成功

### 3. 读取文件内容
```

```
**预期**：调用 `read_file`，显示文件内容

### 4. 编辑文件
```

```

### 5. 递归列出文件

1. **从简单到复杂**：先测试1-6，确保基础功能OK
2. **观察工具调用**：每次都看Agent调用了哪些工具
3. **检查Markdown**：看代码块、列表是否正确渲染
4. **测试切换文件夹**：在不同项目中测试
5. **多轮对话**：测试上下文记忆能力
